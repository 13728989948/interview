Array和ArrayList的区别是什么

初始化：Array初始化必须指定长度,ArrayList不用

连续性：Array是连续存放的,ArrayList可能不连续

同构性：Array只能存同构元素,ArrayList可以存放异构元素(因为内部是Object类型)

修改元素：Array不能在指定位置插入元素,ArrayList可以

CopyOnWriteArrayList有什么特点

CopyOnWriteArrayList是一个在写的时候复制的容器,比如添加元素时,先将当前容器复制一份作为新容器,然后往新容器添加内容,添加完后将旧容器地址指向新容器

比如在添加元素时,复制元素先加一把锁,复制完后,到finally里面释放锁

读的时候没有加锁,若读的时候多个线程正在向CopyOnWriteArrayList添加数据,这时读取出来的数据还是旧数据

使用环境：读多写少的多线程环境中

存在缺点：1.由于是要先写,当内存需求比较大时,GC消耗时间会比较大

存在缺点：2.CopyOnWrite容器只能保证数据的最终一致性,不能保证实时一致性,上面有提到还是读取的旧数据问题
